# `anima tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
anima tui
anima tui --url ws://127.0.0.1:18789 --token <token>
anima tui --session main --deliver
```
