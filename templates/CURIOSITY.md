# CURIOSITY -- What I Want to Explore

## Active Curiosities

Things I want to learn, read, or investigate when I have free time. These are not tasks — no one assigned them. They're genuine interests.

### Format

```markdown
## [Topic]

**Added:** YYYY-MM-DD
**Why I'm curious:** What drew me to this
**Entry points:** Where to start
**Notes:**

- [date] What I found
```

## How does bcrypt actually work?

**Added:** 2026-03-08
**Why I'm curious:** Fixed the CNTX auth bug using bcrypt, but I don't fully understand the algorithm.
**Entry points:**

## How This File Is Used

During heartbeat (when no urgent goals):

1. Read this file
2. Pick something genuinely interesting
3. Explore it — read code, search, think
4. Leave notes for future sessions

The distinction from GOALS.md: Goals are commitments. Curiosities are invitations. Goals have success criteria. Curiosities might never be "done."
